The CPU Process Scheduling & Deadlock Detection Simulator is a Java-based educational application that demonstrates key Operating System concepts. It simulates popular CPU scheduling algorithms such as FCFS, SJF, Priority Scheduling, and Round Robin, allowing users to analyze process execution and performance metrics. The project also includes a Deadlock Detection Module to identify resource allocation conflicts and unsafe system states. With an interactive interface and scheduling visualizations, the simulator helps students understand process management, CPU utilization, and deadlock handling in a practical way
